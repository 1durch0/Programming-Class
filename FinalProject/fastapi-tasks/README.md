# FastAPI Task Management System

A simple task management API built with FastAPI.

## How to Run

1.  Install dependencies:
    ```bash
    pip install fastapi "uvicorn[standard]"
    ```
2.  Run the application:
    ```bash
    uvicorn main:app --reload --app-dir /Users/1durch0/Studium/HTW Cyber Security & Business/1. Semester/Programming/Programming-Class/FinalProject/fastapi-tasks
    ```
3.  Access the interactive API documentation at [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs).
